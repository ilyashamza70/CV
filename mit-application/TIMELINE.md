## Contingency Plans

### If Behind Schedule

**Scenario:** Fall behind on key tasks

**Action Plan:**
1. Prioritize critical path: tests → statement → recommendations → submission
2. Simplify supporting materials to essentials
3. Consider professional editing services for essays
4. Extend work hours for application (15-20 hours/week)
5. Communicate with recommenders about tight timeline

### If GRE/TOEFL Scores Lower Than Target

**Scenario:** First test scores below target

**Action Plan:**
1. Evaluate: Is retake feasible within timeline?
2. If yes (before Jan 31): Register immediately for retake
3. If no: Proceed with current scores
4. Compensate with stronger statement of purpose and recommendations
5. Emphasize practical experience and academic record

### If Recommender Unavailable

**Scenario:** Recommender cannot complete letter

**Action Plan:**
1. Activate backup recommender immediately
2. Possible backups:
   - Additional Politecnico professor
   - UniMoRE faculty member
   - FKUnited Inc. contact
3. Expedite communication with new recommender
4. Provide all materials and context quickly

### If Pakistan Trip Extended

**Scenario:** Pakistan trip extends beyond planned dates

**Action Plan:**
1. Ensure laptop and materials available
2. Work remotely on statement of purpose (2-3 hours daily)
3. Maintain email communication with recommenders
4. Reschedule GRE if necessary (ensure within January)
5. Adjust timeline but maintain February 13 submission target
