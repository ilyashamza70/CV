## File Organization Standards

### Naming Convention
```
[Document_Type]_[Institution/Source]_[Date/Year].pdf

Examples:
- Transcript_UniMoRE_2021.pdf
- GRE_Score_Report_2026-01-10.pdf
- Bachelor_Degree_UniMoRE_2021.pdf
```

### File Quality
- All PDFs must be clear and legible
- Scan resolution: 300 DPI minimum
- File size: <10MB per file (compress if needed)
- Color scans for official documents with seals/stamps

### Backup Strategy
- Primary: This repository folder
- Backup 1: Personal cloud storage (Google Drive, Dropbox)
- Backup 2: Local hard drive
- Backup 3: Email to self (important documents)
