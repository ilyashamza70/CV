# MIT Graduate Application Checklist

## Academic Documents

### Transcripts
- [ ] Official transcript from University of Modena and Reggio Emilia (UniMoRE) - Bachelor's Degree
- [ ] Official transcript from Polytechnic of Turin (Politecnico di Torino) - Master's Degree
- [ ] Transcript from Erasmus semester in Argentina (2024-2025)
- [ ] Transcript from Erasmus period in England
- [ ] WES or ECE credential evaluation (if required for international degrees)

### Degree Certificates
- [ ] Bachelor's degree certificate (Computer Engineering, UniMoRE)
- [ ] Master's degree certificate (Embedded Systems, Politecnico) - *Expected graduation date*
- [ ] Course syllabi for key technical courses (if requested)

## Standardized Test Scores

### Required Tests
- [ ] GRE General Test scores
  - [ ] Register for GRE
  - [ ] Take GRE exam
  - [ ] Send official scores to MIT (institution code: 3514)
- [ ] TOEFL or IELTS scores (international applicants)
  - [ ] Register for TOEFL/IELTS
  - [ ] Take exam
  - [ ] Send official scores to MIT

### Optional Tests
- [ ] GRE Subject Test (Computer Science) - *Optional but recommended*

## Letters of Recommendation

- [ ] Identify 3 recommenders:
  - [ ] **Recommender 1:** Academic advisor/professor from Politecnico di Torino
  - [ ] **Recommender 2:** Bachelor's thesis supervisor from UniMoRE
  - [ ] **Recommender 3:** Professional supervisor from Tomra Recycling
- [ ] Request letters of recommendation (minimum 2 months before deadline)
- [ ] Provide recommenders with:
  - [ ] CV/Resume
  - [ ] Statement of purpose draft
  - [ ] Research interests summary
  - [ ] Deadline information
- [ ] Follow up with recommenders 2 weeks before deadline
- [ ] Confirm submission of all letters in online portal

## Essays and Personal Statements

### Statement of Purpose
- [ ] Draft statement of purpose (2-3 pages)
- [ ] Highlight research interests in embedded systems and IoT security
- [ ] Explain fit with MIT faculty and research groups
- [ ] Discuss career goals and how MIT contributes
- [ ] Review and revise (minimum 3 drafts)
- [ ] Have peers/mentors review
- [ ] Finalize and proofread

### Relevant Experience Description
- [x] Complete 250-word description of relevant experience
  - [x] Cover embedded systems specialization
  - [x] Mention dual Bachelor's (UniMoRE) and Master's (Polito)
  - [x] Include Tomra Recycling and FKUnited Inc. experience
  - [x] Highlight international experience

### Personal Statement (if required)
- [ ] Draft personal statement addressing diversity/background
- [ ] Discuss journey from Pakistan to international engineering career
- [ ] Review and finalize

## CV and Resume

- [ ] Update CV with most recent experience
  - [ ] Education section (Bachelor's and Master's)
  - [ ] Work experience (Tomra Recycling, FKUnited Inc.)
  - [ ] Research experience and thesis projects
  - [ ] Publications (if any)
  - [ ] Technical skills (embedded systems, programming languages)
  - [ ] Certifications
  - [ ] International experience (Erasmus, work visas)
- [ ] Tailor resume for MIT application (2-3 pages maximum)
- [ ] Proofread for errors and formatting

  - [ ] Thesis proposal or abstract
  - [ ] Current progress summary
  - [ ] Expected completion date
- [ ] Portfolio of significant projects

### Professional Certifications

## Application Portal

- [ ] Create MIT application account
- [ ] Complete online application form
  - [ ] Personal information
  - [ ] Educational background
  - [ ] Test scores (self-reported)
  - [ ] Work experience
  - [ ] Research interests
  - [ ] Program selection (Electrical Engineering & Computer Science)
- [ ] Upload all required documents
- [ ] Pay application fee (or request fee waiver)
- [ ] Review entire application for completeness
- [ ] Submit application before deadline

## Post-Submission

- [ ] Confirm receipt of application
- [ ] Track recommendation letter submissions
- [ ] Monitor email for interview requests or additional information requests
- [ ] Prepare for potential interview

  - [ ] Prepare to discuss research interests
  - [ ] Practice common interview questions
- [ ] Send thank-you notes to recommenders

- [ ] I-20 form from MIT
- [ ] Financial documentation for visa
- [ ] Visa interview preparation
- [ ] Travel arrangements to Boston
