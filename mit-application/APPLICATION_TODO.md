## Emergency Contingencies

### If Running Behind Schedule
1. Prioritize critical path items (tests, recommendations, statement of purpose)
2. Consider professional editing services for essays
3. Simplify supporting materials to essentials only
4. Focus on application completion over perfection

### If Recommender Drops Out
1. Have backup recommender identified
2. Can request up to 4 recommendations (submit best 3)
3. Professional references can substitute for academic if necessary

### If Test Scores Lower Than Expected
1. Consider retaking if time permits
2. Emphasize strong academic record in application
3. Highlight practical experience to compensate
